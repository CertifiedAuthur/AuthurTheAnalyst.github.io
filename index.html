Keras Image Classifier using VGG16 pre-trained model compared to a model built using RESNET50

This project focuses on developing and comparing two image classification models using Keras for the specific task of identifying cracks in concrete surfaces. The dataset [https://s3-api.us-geo.objectstorage.softlayer.net/cf-courses-data/CognitiveClass/DL0321EN/data/concrete_data_week4.zip](Click here to Download) consists of images of concrete, where the objective is to classify each image as either positive (cracked) or negative (not cracked). The first model utilizes the VGG16 pre-trained model, which has been pre-trained on large-scale image datasets. The second model is was loaded as it has been built in a previous project, RESNET50 is known for its depth and effectiveness in feature extraction. Both models are trained and fine-tuned using the concrete image dataset, with the aim of accurately distinguishing between cracked and non-cracked concrete surfaces. Evaluation metrics such as accuracy, precision, recall, and F1-score are employed to assess the performance of each model. By comparing the performance of the VGG16 and RESNET50 models, this project aims to determine which architecture is more effective for the task of concrete crack detection. The insights gained from this comparative analysis can inform future efforts in developing image classification systems for detecting cracks in concrete structures.
